Steps

Steps to do that:
  1. Get the icons from somewhere or create them themselves. The example is taken from https://github.com/pychess/pychess.
  2. Add appropriate CSS to your HTML page (here the CSS is contained in chessicons.css).
  3. Use the parameter pieceStyle to add your customized piece set to the viewer. Spelling should be the same as the class added by your CSS.
